Usando links internos e externos

  1. Links externos

    2. Links utilizando a mesma aba do navegador

    É costume usar esse tipo de link quando estamos navegando dentro do mesmo projeto

    Usamos a tag <a>, no exemplo abaixo a palavra "aqui" virará um link para a site da globo usariamos esse comando:

    <a href="https://www.globo.com" > aqui:</a>

    Podemos testar clicanco aqui aqui:

    Note que com esse comando o navegador ira abrir o site da globo na mesma aba que o site inicial, logo sua página será fechada.

    Links utilizando uma outra aba do navegador

    1. Nesse caso usamos dentro da tag <a> os seguintes parâmetros.

      <a href="https://www.globo.com" target = "_blank" > aqui:</a>

      Podemos testar clicando aqui:

    2. Podemos "avisar" o navegador que estamos usando um link externo usando a comando rel = "external", segue o exemplo logo abaixo:

      <a href="https://www.globo.com" target = "_blank" rel = "externo"> aqui:</a>

    Evitar indexação com sites externos

    1. Quando usamos links externos podemos "avisar" o navegador/buscador que damos aval a ele usando a comando
      rel = "nofollow", desse modo evita que o buscador indexe o conteúdo desse site.

      Desse modo temos:

      "<a href="https://www.globo.com" rel = "nofollow" > aqui:</a>"

      Podemos testar clicanco aqui aqui:

  2. Links internos da sua página

    1. Usando o nome do próprio arquivo.

      <a href="pag002.html" > aqui:</a>

      Podemos ir para a segunda página clicando aqui:

    2. Podemos avisar o navegador e sistemas de buscas o direcionamento dos links usando rel = "previous" ou rel = "next" e desse modo o navegador e sistemas de buscas sabem a ordem das páginas dentro de um site com links em cascata.

      Exemplos:

      1. PREVIOUS - <a href="pag002.html" rel = "previous" > aqui:</a>

      2. NEXT - <a href="pag002.html" rel = "next" > aqui:</a>

    3. Usando uma pasta diferente da pasta raiz do projeto:

        4. Quando usarmos uma página diferente temos que colocá-la no endereço.

        Podemos ir para a Terceira página clicando aqui:

      1. Ida:

        2. Exemplo:

        <a href="noticias/pag003.html" rel = "previous" > aqui:</a>

      2. Volta:

        3. Exemplo:

        <a href="../index.html" rel = "next">aqui:</a>

  3. Links para Downloads

    Para criarmos um link para download usamos também a tag <a> e em href colocamos o local do arquivo.

    Note que nesse caso o navegador abre o PDF ao invés de baixá-lo, isso acontece com alguns arquivos, desse modo para podermo, daí temos duas possibilidades:

    • Alguns arquivo como PDF podem abrir no próprio navegador.

      A tag usada para isso será:

      "<a href="livro/meulivro.pdf" > aqui:</a>"

      O resultado que teremos será esse:

      Vc pode testar clicando aqui.

    • Para "obrigar" o navegador baixar o arquivo usados tipos. No caso deo PDF usamos o tipo type = "application/pdf".

      Vc pode testar clicando aqui.

      Podemos encontrar os tipos de cada arquivo no site https://www.iana.org/assignments/media-types/media-types.xhtml

    OBS: Podemos usar href"#", como vc pode ver com "#" no lugar do local do arquivo ou endereço do site, desse modo visualmente eu tenho o link, porém ele leva ao início da página.

    Vc pode testar clicando aqui.
