Técnicas de inserção de CSS3

Podemos inserir o CSS de três maneiras diferentes.
  • In-line
  • Interna
  • Externa

In-line

O método in-line ou em linha é um método onde colocamos os comenados de estilos na mesma linha do código HTML.

Externa

O método externo consiste em colocar os comando de estilos em um documento externo ao documento HTML e chamarmos esse documento na cabeça ,<head>, de cada documento através do link:

<link rel="stylesheet" href="style.css">

Desse modo todos os arquivos HTML que tiverem esse link na cabeça "<head>" terão os mesmos estilos.
