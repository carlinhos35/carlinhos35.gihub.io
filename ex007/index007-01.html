Exemplo de uso da tag <pre>

Cógido usando tag <pre>

           
    <pre>       
        num = int(input('Digite um número')) 
        if num % 2 == 0: 
        print(f'O número {num} é PAR) 
        else:
        print(f'O número {num} é IMPAR)
        print ('Fim do programa')
    </pre>

Segue abaixo a implementação do cógico acima.

         
        num = int(input('Digite um número'))
        if num % 2 == 0:
        print(f'O número {num} é PAR)
        else:
        print(f'O número {num} é IMPAR)
        print ('Fim do programa')

Note que a formatação é igual a usada no escrita do código.
