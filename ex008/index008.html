Trabalhando com listas

Listas ordenadas

Em listas ordenas podemos usar como indexador números, letras e números romanos, ambos em maiúsculo ou minúsculo.

Para isso usamos a tag <ol> (ordered list) e dentro do <ol> usamos o <li> (list item) para cada item.

Para o tipo de indexador e o inicio usamos os comandos: type e start, desse modo teremos:

Note que o inicio (start) é dado em número independentemente do padrão adotado no indexador.

  1. <ol type = "1" start = "4">
    1. Acordar
    2. ligar para João
    3. Tomar café
    4. Escovar os dentes

  2. <ol type = "A" start = "3">
    1. Acordar
    2. ligar para João
    3. Tomar café
    4. Escovar os dentes

  3. <ol type = "A" start = "2">
    1. Acordar
    2. ligar para João
    3. Tomar café
    4. Escovar os dentes

  4. <ol type = "i" start = "2">
    1. Acordar
    2. ligar para João
    3. Tomar café
    4. Escovar os dentes

  5. <ol type = "I">.

    1. Acordar
    2. ligar para João
    3. Tomar café
    4. Escovar os dentes

    Note que coso não é declarado o start, então a contagem começa do primeiro número/letra

Listas não ordenadas

Para listas não ordenadas usamos a tag <ul> (unordered list), e dentro da <ul> também usamos o <li>.

Em listas não ordenadas podemos usar os marcadores "disc", "circle" e "square" e desse modo temos:

  1. <ul type = "disc">

    2.
    • Pão
    • Leite
    • Tomate

  2. <ul type = "circle">

    3.
    • Pão
    • Leite
    • Tomate

  3. <ul type = "square">

    4.
    • Pão
    • Leite
    • Tomate

Mesclagem de listas

Note que podemos colocar <ol> (ordered lists) dentro de <ol>, <ol> dentro de <ul> e vice-versa. Desse modo temos:

  1. Playstation
    2.
    1. PS1
      • Gran Turismo
      • Resident Evil
    2. PS2
    3. PS3
    4. PS4
    5. PS5
  2. xbosta
    3.
    1. xbosta
      • Uma bosta qualquer
      • Outra bosta qualquer
    2. xbosta 360
    3. xbosta one
    4. xbosta series x
    5. xbosta series s
