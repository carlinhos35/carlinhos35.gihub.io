Aprendendo colocar imagens

Imagens intenas

Para inserir uma imagem em usamos a tag <img>, porém temos que indicar o caminho, logo temos: 

<img src="Imagens/PNG/HTML5 - 003 - 200.png" alt="Logo HTML5">

Note que temos o src = source ou origem do arquivo e alt = texto alternativo caso ocorra falha no carregamento.

Imagens externas

Para inserir imagens externas, basta no local de origem (src) colocar o endereço da imagem, exemplo:
