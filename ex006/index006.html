Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em strong em destaque usando a tag <strong> (semântica).

Itálico / ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo com ênfase usando a tag <em>(semântica)

Marcadores de texto

Podemos também criar um texto marcado usando a tag <mark>.

Usando dentro da tag <mark>o comando do CSS3 temos <mark style="background-color::cyan;"> e desse modo o texto ficar marcado com a cor escolhida, nesse caso ciano. Podemos também escolher a cor pelo RGB usando no lugar da cor a proporção RGB.

Exemplo:

<mark style="background-color::rgb(255, 196, 0);">

Para mudarmos a cor de todos marcadores de textos para uma única cor usamos a tag <mark>, porém no <head>escolhemos a cor do background, vide index2.html desse exercício.

Textos grandes e pequenos

Podemo criar um texto grande usando a tag <big>, porém essa tag já está obsoleta.

Podemos criar um texto pequeno usando a tag <small>.

Texto inserido e deletado

Podemos marcar um texto como inserido usando a tag <ins>.

Podemos apenas sublinhar um texto usando a tag <u>, porém essa tag não é semântica

Podemos marcar um texto como deletado usando a tag <del>.

Subescrito e sobrescrito

Usamos subescreito para escrever coisas como a fórmula da água (H2O) usando a tag <sub>.

Usamos subreescrito para escrever coisas como potência em formulas matemáticas (ab+c) usando a tag <sup>.
