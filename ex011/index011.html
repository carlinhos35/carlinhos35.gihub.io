Inserindo vídeos hospedados localmente

Formatos de vídios usados

Assim como pode acontecer com audios, existem alguns navegadores que não abrem determinados formatos de vídeos.Logo, assim como nos audios e fotos, posso criar uma lista de prioridades com o mesmo vídeo em vários formatos.

Formatos suportados

Os formatos suportados pela maioria dos navegadores são:

  • Arquivos .mp4
  • Arquivos .m4v
  • Arquivos .webm
  • Arquivos .ogv

Exemplo de códigos para inserção de vídeos


                    <video width = "250" poster = "imagens/tumb-cafe-200x133.jpg" control>>
                        <source src = "midia/Meu-Video-webM.webm" type = "video/webm">
                        <source src = "midia/meu-video-mp4.mp4" type = "video/mp4">
                        <p>Seu navegador não em suporte para reprodução desses vídeos</p>
                    </video>

Para arquivos m4v e ogv usamos, repectivamente, os types video/mp4 e video/ogg.

O código acima implicará no vídeo abaixo. Note que o cógido tem mais de um formato de vídeos.

Inserindo vídeos do youtube

Para aprender como copiar o código direto do youtube click aqui:

Segue um exemplo de cógido copiado

<iframe width="420" height="236.25" src="https://www.youtube.com/embed/eYGjnuK8nYo" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

Segue abaixo o vídeo usando o código acima.

Caso queria que o vídeo começe de um determinado tempo usamos ?start=70, onde 70 é o números, em segundos, que o vídeos começará.

Exemplo:

<iframe width="420" height="236.25" src="https://www.youtube.com/embed/eYGjnuK8nYo?start=70" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
